HOW TO RELEASE
===============================
Update version number in pom.xml, version.html, MultiController.java etc.
Update changelog.jsp, download.jsp
mvn clean install -P full
cd subsonic-assembly
mvn assembly:assembly
Deploy
Check that correct local version is displayed.
Commit
Upload files to SourceForge (in separate shell, REMEMBER TO CHANGE VERSION BELOW).
  rsync -avP -e ssh subsonic-installer-windows/target/subsonic*-setup.exe  sindre_mehus,subsonic@frs.sourceforge.net:/home/frs/project/s/su/subsonic/subsonic/4.2/
  rsync -avP -e ssh subsonic-assembly/target/subsonic* sindre_mehus,subsonic@frs.sourceforge.net:/home/frs/project/s/su/subsonic/subsonic/4.2/
  rsync -avP -e ssh subsonic-installer-mac/target/subsonic*.pkg subsonic-installer-debian/target/subsonic*.deb subsonic-installer-rpm/target/subsonic*.rpm sindre_mehus,subsonic@frs.sourceforge.net:/home/frs/project/s/su/subsonic/subsonic/4.2/

Modify FRS properties
  https://sourceforge.net/project/admin/explorer.php?group_id=126265
