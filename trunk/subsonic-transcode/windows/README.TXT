==========================================================
            SUBSONIC TRANSCODING PACK
==========================================================

This is a collection of free audio codecs for Windows.
To be installed in c:\subsonic\transcode

See http://subsonic.sourceforge.net/transcode for details.

Sindre Mehus, Aug 03 2006
==========================================================

oggdec
Ogg Vorbis decoder
Version 1.9.2
http://www.rarewares.org/ogg.html

oggenc2
Ogg Vorbis encoder
Version 2.83 using libVorbis v1.1.2
http://www.rarewares.org/ogg.html

flac
FLAC (Free Lossless Audio Codec) encoder/decoder
Version 1.1.2
http://flac.sourceforge.net

faad
AAC/MP4 decoder
Build Jul 9 2004
http://www.rarewares.org/aac.html

lame
MP3 encoder
Version 3.96.1
http://www.rarewares.org/mp3.html

wmadec
WMA decoder
http://svn.slimdevices.com/tags/RELEASE_6_0_2/tools/wmadec/#dirlist

mac
APE (Monkey's Audio) decoder
Version 2.99
http://www.rarewares.org/ogg.html

mppdec
MPC (Musepack) decoder
Version 1.95e
http://www.musepack.net/index.php?pg=win

ofr
OptimFROG Lossless decoder/encoder
Version 4.509
http://www.losslessaudio.org/

ofs
OptimFROG DualStream decoder/encoder
Version 4.509
http://www.losslessaudio.org/

shorten (incl cygwin1.dll)
Shorten (shn) decoder
Version 3.6.0
http://www.etree.org/shnutils/shorten/

wvunpack
WavPack decoder
Version 4.31
http://www.wavpack.com/downloads.html
