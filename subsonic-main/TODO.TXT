HOW TO RELEASE
-------------
Update version number in pom.xml etc.
mvn clean install
mvn assembly:assembly
mv target/subsonic-2.7-bin.zip target/subsonic-2.7.zip
Deploy
Check that correct local version is displayed.
Commit
Create file release at SourceForge.
mvn site:site
mvn site:deploy
svn copy https://svn.sourceforge.net/svnroot/subsonic/trunk https://svn.sourceforge.net/svnroot/subsonic/tags/release-2.7.beta1 -m "Tagging 2.7.beta1."
Register at freshmeat.
Update version number in pom.xml etc.

------------------

wma -> mp3  wma           mp3         wmadec -w     lame -Q - -
ogg -> mp3  ogg           mp3         oggdec


Player config:
Enabled Transcoder
  x     wma->mp3

o All transcoders *must* be installed in SUBSONIC_HOME/transcoders/
o Use Commons IO.
